BASES DE DATOS

¿Qué es una base de datos?

Una base de datos es una colección de registros integrados y clasificados. Un registro puede ser un nombre, un numero o valor, etc.

Tipo de base de Datos

SQL o bases clásicas

  • Relacionales
  • Se implementa mediante tablas
  • Se presenta en esquemas

NoSQL o bases no relacionales

  • Usa JavaScript
  • No es obligatorio una relacion entre las tablas
  • Se presenta en colecciones de documentos, que son objetos JSON

Consejos


  • Escoge el area en la que te vas a enfocar y averigua que tipo de base de datos es más efectiva.
  • Las bases de datos SQL se usan en aplicaciones bancarias, transaciones en linea, etc.
  • Las bases de datos NoSQL se usan en bases de datos CLAVE - VALOR.
